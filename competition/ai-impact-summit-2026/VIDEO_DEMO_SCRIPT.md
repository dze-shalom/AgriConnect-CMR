## VIDEO SPECIFICATIONS

**Format:** MP4 (H.264 codec, AAC audio)
**Resolution:** 1920x1080 (Full HD) or 1280x720 (HD minimum)
**Frame Rate:** 30 fps
**File Size:** <200 MB
**Aspect Ratio:** 16:9 (landscape)

**Production Tools:**
- **Screen Recording:** OBS Studio (free), Loom, QuickTime (Mac), Windows Game Bar
- **Video Editing:** DaVinci Resolve (free), iMovie, Canva Video
- **Audio:** External microphone (recommended) or quiet environment
- **Music:** Royalty-free background music (low volume): YouTube Audio Library, Epidemic Sound

## PRODUCTION CHECKLIST

### Pre-Production
- [ ] Write and finalize script (done above)
- [ ] Gather stock footage (farmers, fields, crops)
  - Sources: Pexels, Unsplash, Pixabay (free)
  - Keywords: "African farmer", "Cameroon agriculture", "maize field"
- [ ] Record dashboard screen capture (OBS Studio)
  - Clean up browser (close extra tabs, full screen)
  - Use demo data if live data is messy
- [ ] Prepare farmer testimonial (video or voiceover)
- [ ] Download royalty-free music (YouTube Audio Library)
- [ ] Create graphics:
  - AgriConnect logo (high-res PNG)
  - Impact metrics infographic (Canva)
  - On-screen text slides

### Recording
- [ ] **Dashboard Screen Recording:**
  - Resolution: 1920x1080 (match final video)
  - Frame rate: 30 fps
  - Cursor visibility: ON (shows interaction)
  - Audio: OFF (narration added later)
  - Duration: Record 1-2 minutes (edit to 30 seconds)

- [ ] **Narration Audio:**
  - Microphone: External USB mic (Blue Yeti, ATR2100) or quiet built-in
  - Environment: Quiet room, minimal echo
  - Script reading: Practice 3x before recording
  - Record 2-3 takes (choose best)
  - Export as WAV or high-quality MP3

- [ ] **Farmer Testimonial:**
  - If video: Smartphone (1080p), landscape mode, stable (tripod or prop)
  - Lighting: Outdoor (shade) or indoor (window light)
  - Audio: Close to subject, minimize background noise
  - Duration: 20-30 seconds

### Post-Production (Video Editing)
- [ ] **Import Assets:**
  - Stock footage (intro/outro)
  - Dashboard screen recording
  - Farmer testimonial
  - Narration audio
  - Background music

- [ ] **Video Timeline (DaVinci Resolve / iMovie):**
  - 0:00-0:20: Stock footage + text overlays
  - 0:20-0:50: Dashboard screen recording
  - 0:50-1:20: AI features screen recording
  - 1:20-1:50: Alerts demo + impact infographic
  - 1:50-2:10: Farmer testimonial
  - 2:10-2:30: Call to action (logo, text)

- [ ] **Audio Mix:**
  - Narration: Center, -3dB to -6dB peak
  - Background music: -20dB to -25dB (subtle, don't overpower voice)
  - Farmer audio: Match narration level
  - Use audio compressor for consistent volume

- [ ] **Text Overlays:**
  - Font: Clean sans-serif (Montserrat, Roboto, Open Sans)
  - Size: Large enough to read on mobile (48pt+)
  - Color: White with black shadow/outline (readability)
  - Animation: Fade in/out (1 second transitions)

- [ ] **Color Grading:**
  - Slightly warm tones (inviting, agricultural)
  - Consistent across all clips
  - Avoid oversaturation

- [ ] **Subtitles/Captions:**
  - **HIGHLY RECOMMENDED** for accessibility
  - Use auto-caption tools (YouTube, Descript) then manually correct
  - Export as burned-in subtitles or separate SRT file

### Export Settings
- [ ] **Format:** MP4
- [ ] **Video Codec:** H.264 (most compatible)
- [ ] **Audio Codec:** AAC
- [ ] **Resolution:** 1920x1080 (1080p) or 1280x720 (720p min)
- [ ] **Frame Rate:** 30 fps
- [ ] **Bitrate:** 5-10 Mbps (video), 128-192 kbps (audio)
- [ ] **File Size:** Check <200 MB (reduce bitrate if over)

### Quality Control
- [ ] Watch full video 3x (catch errors)
- [ ] Check audio levels (no clipping, consistent volume)
- [ ] Verify all text is readable (especially on mobile)
- [ ] Test on multiple devices (desktop, tablet, phone)
- [ ] Confirm timing (exactly 2:20-2:30, not over)
- [ ] Check file size (<200 MB)
- [ ] Spellcheck all on-screen text

### Distribution

  - Title: "AgriConnect - AI for All Global Impact Challenge Demo"

## ALTERNATIVE: SIMPLE VERSION (If Limited Resources)

If you can't produce a full cinematic video, a **simpler but effective version**:

**Equipment Needed:**
- Smartphone with good camera (1080p)
- Free video editor (CapCut, InShot, iMovie)
- Quiet room

**Structure:**
1. **0:00-0:30:** You presenting to camera (introduction, problem statement)

3. **1:30-2:00:** Back to camera (impact metrics, testimonial quote)
4. **2:00-2:30:** Call to action, show contact info

**Tips:**
- Look at camera (not screen)
- Good lighting (face camera toward window or use lamp)
- Speak clearly, enthusiastically
- Use cue cards (place behind camera)
- Smile! (passion is contagious)

## FINAL TIPS FOR SUCCESS

1. **Keep It Simple:** Don't overcomplicate. Clear narration + good screen recording > fancy effects.

2. **Show, Don't Just Tell:** Demonstrate the dashboard actually working (live data updates, clicking features).

3. **Emphasize AI:** Judges want to see AI in actionâ€”spend time on ML predictions, not just sensor readings.

4. **Humanize It:** Farmer testimonials create emotional connection (critical for impact-focused challenges).

5. **Professional Audio:** Bad audio ruins good video. Invest time in clear narration.

6. **Test on Mobile:** Many judges may watch on phonesâ€”ensure text is readable.

7. **Call to Action:** End strong with vision and contact info (make it easy to follow up).

8. **Authenticity Over Perfection:** A genuine, passionate demo beats a soulless corporate video.
