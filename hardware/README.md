# AgriConnect Hardware

This directory contains firmware for field nodes and gateways.

## Components

### Field Nodes
- ESP32 DevKit
- LoRa SX1276 module
- Multiple sensors (DHT22, soil moisture, etc.)
- See `field_node/` directory

### Gateway
- ESP32 DevKit
- LoRa SX1276 module
- WiFi connectivity
- See `gateway/` directory
